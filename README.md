# AI Emotion Detector API

An intelligent web application built with **Flask** and **Machine Learning** that analyzes text input to predict the underlying emotion (Joy, Anger, Sadness, etc.) along with relevant emojis.

## Features
- **Real-time Prediction:** Get instant results using an optimized ML model.
- **Fully Responsive UI:** Optimized for Desktop, Tablet, and Mobile (Portrait mode).
- **API Support:** Can be integrated with Android/Flutter apps via JSON responses.
- **Clean Interface:** User-friendly design with a dedicated sidebar and result display.
